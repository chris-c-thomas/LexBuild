- **`law2md index` command**: Skipped. _meta.json is generated during convert; standalone rebuild is a niche use case.

- **CI matrix**: Node 20 and 22 only (Node 18 EOL April 2025).
- **Publish workflow**: Uses `changesets/action@v1` with `NPM_TOKEN` secret. Provenance via `id-token: write` permission.

- Publishes: `@law2md/core`, `@law2md/usc`, `law2md` (CLI with `bin` entry)
